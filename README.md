# Chuangshinxin Bakery B2B Website

This is a single-page static website for **創食新 烘焙坊**, designed for B2B wholesale business.

## Features
- **Responsive Design**: Built with Tailwind CSS, works on mobile and desktop.
- **SEO Optimized**: Includes meta tags for title, description, keywords, and Open Graph.
- **Sections**: Hero, About Us, Products, Customization Service, Contact Form.
- **Contact Form**: Configured to use Formspree (requires setup).

## Setup & Deployment

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/chuangshinxin-bakery-b2b.git
    ```

2.  **Customize**:
    -   Replace placeholder images in `index.html` with actual assets in `images/`.
    -   Update the Formspree URL in the `<form>` tag in `index.html`.
    -   Update the `og:image` URL in `<head>`.

3.  **Deploy to GitHub Pages**:
    -   Go to repository Settings > Pages.
    -   Select `main` branch as source.
    -   Save.
